# Contact Management System

This project is developed as part of the Prodigy InfoTech Software Development Internship (Task-03).

## Description
The Contact Management System is a simple Python-based application that allows users to manage contact details efficiently. Users can add, view, update, and delete contacts through a simple interface.

## Features
- Add new contacts
- View saved contacts
- Update existing contact information
- Delete contacts
- Store basic details such as name, phone number, and email

## Technologies Used
- Python

## How to Run the Program
1. Install Python on your system.
2. Download or clone this repository.
3. Open the project folder.
4. Run the Python file using:
   python contact_management.py
