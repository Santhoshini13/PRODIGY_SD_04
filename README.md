# Sudoku Solver

This project is developed as part of the Prodigy InfoTech Software Development Internship.
## Description
The Sudoku Solver is a Python-based program that automatically solves a Sudoku puzzle using an algorithmic approach. It uses logical techniques such as backtracking to fill the missing numbers while following Sudoku rules.
## Features
- Solves standard 9x9 Sudoku puzzles
- Uses backtracking algorithm
- Displays the solved Sudoku grid
- Demonstrates problem-solving and algorithm implementation
## Technologies Used
- Python
## How to Run the Program
1. Install Python on your system.
2. Download or clone this repository.
3. Open the project folder.
4. Run the Python file using the command:
   python sudoku_solver.py
